//将数据读入存放在一维数组a[n]中
//进行n-1躺
for(int i=1;i<n;i++){
//第i躺时从a[0]到a[n-1-i]都与他们下一个数比较
 for(int j=0;j<n-i;j++){
 //本质是交换
  if(a[j]>a[j+1]){
   //如果左边的数更大，则交换
     int temp=a[j];
     a[j]=a[j+1];
     a[j+1]=temp;
    }
  }
}

总结：n个数，要进行n-1躺排序，而每趟排序中，从a[0]到a[n-1-i]都需要与下一个数比较
